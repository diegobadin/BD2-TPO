# Trabajo Práctico Obligatorio
## 2025 - 1Q

Para ejecutar el sistema correr en una terminal bash desde la raíz del projecto:
```
cd server
npm start
```
Luego en otra terminal bash desde la raíz del projecto:
```
cd client
npm start
```
